## Discord (powershellyoungteam)

> **PSExchangeClock v1.0.0 is live!** 🎉
>
> Remember that wee LSE countdown timer I made a while back? It's grown into a full global markets dashboard and I've finally packaged it up as a proper PowerShell module.
>
> **What it does:**
> • Live countdown timers for 20 major stock exchanges (NYSE, LSE, Tokyo, etc.)
> • Interactive world map with NASA Earth at Night imagery
> • Toggleable map overlays — earthquakes, volcanoes, conflict zones, power plants, submarine cables, political borders, and more
> • Overlay legend for conflict zones, power plants, and volcano alert levels
> • World clocks across all major time zones
> • Market data sidebar — financial news, forex rates, crypto prices, stock indices, commodities, individual stock quotes
> • Desktop notifications before exchange close
> • Secure API key storage via Windows Credential Manager
> • Dark-themed WPF UI with sortable exchange grid
>
> **Install it:**
> ```powershell
> Install-Module PSExchangeClock -Scope CurrentUser
> Start-PSExchangeClock
> ```
>
> News, forex, and crypto work straight away. For indices/commodities/stock quotes, grab free API keys from Twelve Data and Alpha Vantage.
